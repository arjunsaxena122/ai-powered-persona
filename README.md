# AI Powered Persona

A project to create and interact with AI-powered personas.

## How to Set up Locally

1. Clone the repository

```bash
git clone https://github.com/arjunsaxena122/ai-powered-persona.git
```

2. Go to the folder

```bash
cd ai-powered-persona
```

3. Install packages

- Go to the <b>ai-powered-persona</b> folder and run the following command in the terminal:

```bash
pnpm i
```

4. Run the server locally

- Go to terminal and write these following commands

```bash
pnpm run dev
```

## Environment variables

1. For Gemini AI

```bash

GEMINI_API_KEY=your-gemini-ai-api-key

GEMINI_BASE_URL=https://generativelanguage.googleapis.com/v1beta/openai

```

2. For OpenAI

```bash
OPENAI_API_KEY=your-openai-api-key
```
